
## Installation
Run `npx install-peerdeps -r https://npm.pkg.github.com/yello-io/ -d @yello-io/eslint-config-yello-base` in your project.


## Adding a new file to rules directory
Use the following template:

```javascript 
module.exports = {
	plugins: [],
	env: {},
	parserOptions: {},
	rules: {},
	settings: {},
};
```
